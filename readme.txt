UM Import CPT Data helps you easily imports csv file to directory custom posts.

== Description ==

UM Import CPT Data turns WordPress sites into a fully-fledged content management system by giving you all the tools to do more with your data.

UM Import CPT Data plugin to help csv files import to create directory posts with below fields.

1. Name (Post Title)
2. Image (Featured Image)
3. Category (Taxonomy Of Post type Multiple category with comma separated values)
4. Description (Post Content)
5. Gallery Images (Multiple Images with Comma separated values store in ACF Gallery)
6. Type (Internal, External (Radio Buttons) ACF Field)
7. User Name (Post Author, User added in WP Backend with Editor Role)
8. User email (User added in WP Backend with Editor Role)


== Step to follow ==

1. Install ACF pro plugin and create one group and add two fields as below.
	a. Gallery Images (ACF gallery field)
	b. Type (ACF radio field) with two options Internal and External
2. Upload CSV file with formated column (Please check plugin root sample format csv file).
